# VR Exhibition - Udhëzime të Shpejta

## 🚀 Fillimi i Shpejtë

### Hapi 1: Instaloni Node.js

- Shkarkoni nga: https://nodejs.org/
- Instaloni versionin LTS (18+)

### Hapi 2: Ekzekutoni Instaluesin

- Klikoni dy herë në `install.bat` (për CMD)
- Ose `install.ps1` (për PowerShell)

### Hapi 3: Hapni Projektin

- Shkoni te: http://localhost:3000
- Përdoruesi: http://localhost:3000/user
- Admin: http://localhost:3000/admin

## 📋 Çfarë bën instaluesi automatik?

1. ✅ Kontrollon Node.js dhe npm
2. 📦 Instalojnë të gjitha dependencat
3. 🎯 Tregojnë udhëzime për përdorim
4. 🚀 Nisën serverin e zhvillimit

## 🔧 Probleme të Mundshme

### "Node.js nuk është i instaluar"

- Shkarkoni nga nodejs.org
- Instaloni dhe rinisni kompjuterin

### "npm install dështon"

- Kontrolloni lidhjen internet
- Fshini `node_modules` dhe provoni përsëri

### "Porti 3000 është i zënë"

```bash
npm run dev -- -p 3001
```
