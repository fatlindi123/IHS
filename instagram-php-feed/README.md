# Instagram PHP Feed

This example shows the latest 6 Instagram posts using:

- `PHP`
- `JavaScript`
- `CSS Flexbox`

## Setup

1. Open `instagram-feed.php`
2. Replace:
   - `YOUR_INSTAGRAM_ACCESS_TOKEN`
   - `YOUR_INSTAGRAM_USER_ID`
3. Upload the folder to a PHP-enabled server
4. Open `index.html`

## Files

- `index.html`: page markup
- `styles.css`: square grid layout with Flexbox
- `script.js`: fetches posts from PHP
- `instagram-feed.php`: calls the Instagram API
